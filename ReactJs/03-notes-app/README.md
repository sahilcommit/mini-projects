# Sticky Notes App

A functional **CRUD** (Create, Read, Update, Delete) application. This is where I mastered the "Heart of React"—State Management.

## 🚀 Key Features
* **Two-Way Binding:** Syncing input fields with React state in real-time.
* **State Updates:** Adding and removing items from a list without mutating the original array.
* **Visual Flair:** A "Sticky Note" theme with custom CSS and hover animations.

## 🛠️ Concepts Learned
* `useState` Hook (Strings and Arrays)
* Form Handling (`onSubmit`)
* Immutability (using the Spread Operator `[...]`)

## ⚙️ Setup
1. `cd 03-notes-app-react`
2. `npm install`
3. `npm run dev`
