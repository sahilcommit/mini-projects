# React Multi-Page Navigation

A full-scale Multi-Page Application (MPA) simulation. This project handles the complexity of app architecture and URL management.

## 🚀 Key Features
* **Dynamic Routes:** Using `:id` parameters to load specific content (e.g., Course Details).
* **Nested Layouts:** Using `<Outlet />` to render sub-pages inside a parent layout.
* **Programmatic Navigation:** Navigating the user via buttons using `useNavigate`.

## 🛠️ Concepts Learned
* `BrowserRouter`, `Routes`, and `Route`
* `useParams` for URL data
* Handling 404 (Not Found) errors

## ⚙️ Setup
1. `cd 05-react-router-project`
2. `npm install`
3. `npm run dev`
