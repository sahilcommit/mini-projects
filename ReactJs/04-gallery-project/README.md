# Lumina Gallery (API-Driven)

A dynamic image gallery that communicates with an external server. This project bridges the gap between frontend and backend.

## 🚀 Key Features
* **Axios Integration:** Fetching high-resolution image data from the *Lorem Picsum API*.
* **Pagination System:** Moving through hundreds of images using "Prev" and "Next" logic.
* **Conditional Rendering:** Showing "Loading..." states while data is being fetched.

## 🛠️ Concepts Learned
* `useEffect` for Side Effects
* Asynchronous Programming (Async/Await)
* Dependency Arrays in Hooks

## ⚙️ Setup
1. `cd 04-gallery-project-react`
2. `npm install`
3. `npm run dev`
