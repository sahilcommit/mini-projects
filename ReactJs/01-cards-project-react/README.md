# Job Openings Gallery - React

A dynamic project focusing on the power of **Props** and **Data Mapping**. This UI displays job listings from a structured data array.

## 🚀 Key Features
* **Component Reusability:** A single `Card` component used across multiple data points.
* **Dynamic Rendering:** Uses `.map()` to transform an array into a visual grid.
* **Modern UI:** Clean card layout with hover effects and brand logos.

## 🛠️ Concepts Learned
* Functional Components
* Props & Destructuring
* Rendering Lists with unique `keys`

## ⚙️ Setup
1. `cd 01-cards-project-react`
2. `npm install`
3. `npm run dev`
