# Prospective Customer Segmentation UI

A high-fidelity, responsive landing page interface built using **Tailwind CSS** and React. This project demonstrates advanced layout skills.

## 🚀 Key Features
* **Tailwind Utility Classes:** Rapid UI building without leaving the HTML.
* **Horizontal Scroll:** Custom-styled slider for customer profiles.
* **Component Splitting:** Breaking a large page into `Navbar`, `Hero`, and `Card` components.

## 🛠️ Concepts Learned
* Tailwind CSS Configuration (v4/Vite)
* Absolute Positioning & Z-index in React
* Complex Flexbox & Grid layouts

## ⚙️ Setup
1. `cd 02-ui-project-tailwind`
2. `npm install`
3. `npm run dev`
